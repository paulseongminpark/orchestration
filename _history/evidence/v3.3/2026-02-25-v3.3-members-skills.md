## 외부 CLI (v3.3 신규 섹션)

### Codex CLI (OpenAI)

| 항목 | 값 |
|------|-----|
| 버전 | v0.104.0 |
| 모델 | gpt-5.3-codex |
| 플랜 | ChatGPT Plus ($20/월) |
| 제한 | 5시간 롤링 (~30-150 메시지) |
| reasoning | xhigh (기본), 프로필별 조절 |
| 역할 | 정밀 검증기 (diff 리뷰, 포맷 QA, git 추출) |
| 세션 권장 | 3~5회 |

**설정 파일:**

| 파일 | 위치 | 설명 |
|------|------|------|
| instructions.md | ~/.codex/ | 글로벌 페르소나 (추출 전용, _meta 강제, 해석 금지) |
| config.toml | ~/.codex/ | 프로필 3종 + 기본 설정 |
| extract-git-history.md | ~/.codex/prompts/ | Git 히스토리 → JSON 추출 |
| review-diff.md | ~/.codex/prompts/ | uncommitted diff → 이슈 JSON |
| content-qa.md | ~/.codex/prompts/ | tech-review 콘텐츠 5관점 QA |

**프로필:**

| 프로필 | reasoning | 용도 |
|--------|-----------|------|
| extract | high | 구조화 데이터 추출 (정확도 우선) |
| verify | low | completeness 체크 (속도 우선) |
| review | medium | diff 기반 코드 리뷰 (균형) |

### Verify Barrier (3단계)

| 단계 | 내용 | 비용 |
|------|------|------|
| 1. 구조 검증 | JSON 파싱 + _meta 존재 + files_scanned 수치 확인 | ~0 |
| 2. 스팟체크 | 임의 2~3개 항목 원본 파일 대조 | ~1K 토큰 |
| 3. 반박 검증 | "빠진 것 없는가?" (고위험 작업만) | ~3K 토큰 |
